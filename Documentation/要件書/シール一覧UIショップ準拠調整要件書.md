# シール一覧UIショップ準拠調整要件書

## 背景と目的
- 現在の HUD 左下 `シール一覧` パネルは、上部ヘッダーのみ濃いピンク、一覧本体は薄いピンクの2段構成になっている。
- 一方で `シールショップ` は、パネル全体に濃いピンクの外側コンテナを持ち、その内側に薄いピンクのコンテンツ領域を敷く構成になっている。
- 今回は `シール一覧` の見た目を `シールショップ` に揃え、画面内のピンク系 UI 表現を統一する。

## スコープ内
- HUD 左下 `シール一覧` パネルの背景構造、配色、余白、角丸表現の見直し
- `シールショップ` と同系統の「濃いピンク背景 + 薄いピンク下敷き」構成への変更
- 既存のタイトル表示、空表示、スクロール領域、シールセル表示を維持した上での見た目調整
- `HudScreen.uxml` / `HudScreen.uss` と `HubScreenBinder` の参照整合確認

## スコープ外
- シール一覧の選択ロジック、配置ロジック、残数計算ロジックの変更
- シールショップ UI 自体の再調整
- シール一覧セルの情報設計全面変更
- 新規アニメーション、SE、演出の追加
- フェーズ遷移、ショップ購入、妖精画面の仕様変更

## 機能要件

### 1. パネル外観
- `シール一覧` パネル全体は、`シールショップ` の `#sticker-shop-panel` と同系統の濃いピンク背景を持つコンテナとして表示する。
- パネル内部には、`シールショップ` の `#sticker-shop-content-frame` と同系統の薄いピンク領域を配置する。
- 現状の「ヘッダーだけ濃いピンク」という見え方は廃止し、パネル全体として濃いピンクの外枠を感じられる見た目にする。
- 角丸、内側余白、上下の余白感は `シールショップ` と並べたときに同一シリーズと認識できる水準に揃える。

### 2. タイトル領域
- `シール一覧` タイトルは引き続きパネル上部に表示する。
- タイトル文字色は白系を維持し、濃いピンク背景上で十分な視認性を確保する。
- タイトル領域は独立した帯として残してもよいが、帯だけが別色に見える構成にはしない。
- タイトル周辺の上下左右余白は `シールショップ` ヘッダーと近い密度に調整する。

### 3. コンテンツ領域
- 一覧本体はタイトルの下に薄いピンクの下敷きとして表示する。
- 薄いピンク領域の内側に `empty-sticker-list-label` と `sticker-scroll-view` を収める。
- 一覧件数が多い場合でも、薄いピンク領域の中でスクロールし、濃いピンク外側コンテナからはみ出さない。
- シール未所持時は空表示メッセージを表示し、背景構造だけが崩れた見た目にならない。

### 4. 既存セルとの整合
- `sticker-scroll-view` 内のシールセル生成方法は現行どおり `HubScreenBinder.BuildStickerList()` を使用する。
- 各シールセルの選択状態、画像表示、所持数バッジの既存挙動は維持する。
- パネル背景変更後も、シールセルと所持数バッジが背景に埋もれず視認できるコントラストを保つ。

### 5. 既存実装との整合
- `HubScreenBinder` が参照している `bottom-left-sticker-panel`、`sticker-scroll-view`、`empty-sticker-list-label` の名前は原則維持する。
- レイアウト調整のために UXML 構造を変更する場合でも、Binder 側の参照切れを発生させない。
- 見た目調整の主変更点は USS に集約し、ロジック変更は必要最小限に留める。

### 6. 画面内バランス
- `シール一覧` パネルは引き続き HUD 左下に固定表示する。
- `シールショップ` を開いていない通常時に、左上の所持金パネル、左下のシール一覧、右下の各ボタンとの視覚バランスが崩れないこと。
- 16:9 前後の解像度でも、パネルの濃いピンク外枠と薄いピンク下敷きの関係が崩れないこと。

## 非機能要件
- UI 実装は既存どおり UI Toolkit を使用する。
- 色、角丸、余白は USS に定義し、`HubScreenBinder` に見た目定数を書き込まない。
- 既存のショップ配色と比較しやすい命名で USS を整理する。
- 変更後も null 参照例外、要素参照失敗、スクロール不能状態を発生させない。
- 既存プレハブ、ScriptableObject、データソース API は変更しない。

## 受け入れ条件
- 通常 HUD 表示時、左下 `シール一覧` パネル全体が濃いピンクの外側コンテナとして見える。
- `シール一覧` タイトルの下に、薄いピンクの一覧領域が下敷きとして表示される。
- `シール一覧` の配色と余白感が、`シールショップ` の外側パネルとコンテンツ領域の関係に近い見た目になっている。
- シール一覧のスクロール、空表示、選択表示、所持数バッジ表示の既存動作が維持される。
- ショップ開閉、シール配置、フェーズ切り替えを行っても `シール一覧` UI が崩れない。

## 未確定事項・確認事項
- `シールショップ` と完全に同一の色コードを使うか、近似色で HUD 側に最適化するかは実装時に比較確認する。
- `シール一覧` ヘッダー帯を視覚的にどこまで残すかは、ショップとの統一感を優先して最終調整する。
